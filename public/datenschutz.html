2. Grundprinzip: Hybride Datenverarbeitung

Die App "vCardAI" verfolgt einen "Privacy-First"-Ansatz. Standardmäßig werden alle von Ihnen eingegebenen Daten (Texte, Kontakte, Historie) ausschließlich lokal in Ihrem Browser (Local Storage) gespeichert.

Eine Übertragung von Inhaltsdaten an externe Server findet nur dann statt, wenn Sie explizit eine der optionalen KI-Funktionen ("AI Optimieren", "Scannen", "Anreichern") durch Klick auf den entsprechenden Button auslösen.

4. Nutzung von KI-Funktionen (Google Gemini API)

Diese Anwendung bietet Funktionen zur intelligenten Textanalyse, Datenextraktion aus Bildern (OCR) und Datenanreicherung. Hierfür können Sie entweder den Dienst Google Gemini oder ein lokales LLM (z.B. Ollama) verwenden.

4.1. Google Gemini (Cloud)

Wenn Sie Google Gemini als Anbieter wählen, wird die Schnittstelle (API) von Google verwendet.

4.1.1. Art der verarbeiteten Daten (Google)

Wenn Sie eine KI-Funktion (z.B. Button "AI" oder "Scan & Weiter") betätigen, werden folgende Daten an die Google-Server übermittelt:

  • Der von Ihnen eingegebene oder im Editor befindliche Text (z.B. E-Mail-Signaturen).
  • Die von Ihnen hochgeladenen Fotos von Visitenkarten.
  • Der von Ihnen bereitgestellte API-Schlüssel (API Key), sofern Sie diesen in den Einstellungen hinterlegt haben.

4.1.2. Rechtsgrundlage und Einwilligung

Die Übermittlung dieser Daten erfolgt auf Basis Ihrer freiwilligen Einwilligung (Art. 6 Abs. 1 lit. a DSGVO), die Sie durch das aktive Auslösen der Funktion (Klick auf den Button) und das Hinterlegen eines API-Schlüssels erteilen. Sie sind nicht verpflichtet, die KI-Funktionen zu nutzen; die App ist auch rein manuell/lokal voll funktionsfähig.

4.1.3. Datenübermittlung in Drittländer (USA)

Durch die Nutzung der Gemini API werden Daten an Server von Google LLC in den USA übertragen. Google ist unter dem EU-US Data Privacy Framework (DPF) zertifiziert. Die Europäische Kommission hat festgestellt, dass die USA für zertifizierte Unternehmen ein angemessenes Datenschutzniveau bieten.

Wenn Sie ein lokales LLM (z.B. Ollama, LM Studio) konfigurieren, werden Ihre Daten (Texte, Bilder) nicht an externe Server gesendet. Die Verarbeitung erfolgt ausschließlich innerhalb Ihres eigenen Netzwerks (z.B. auf Ihrem Computer unter localhost).

Hinweis: Sie sind selbst dafür verantwortlich, dass Ihr lokaler LLM-Server sicher konfiguriert ist und keine Daten ungewollt nach außen sendet.
